## 分类口径

| 分类 | 收录范围 |
|---|---|
| 开发板 | LaunchPad、评估板、扩展板，以及针对 STM32、Arduino、树莓派、Jetson、RDK X3、MSPM0 等主板的例程 |
| 芯片 | MCU/驱动芯片数据手册、芯片相关板卡资料，以及以 AT8236 等芯片为核心的模块资料 |
| 模块 | 电机驱动、云台、舵机、调试探针等模块的说明、原理图、接线和例程 |
| 比赛规则 | 赛题、参赛报告、器材清单及题面内附的参赛注意事项；分类名不表示每份文件都是完整规则 |

所有 PDF 至少属于一个分类。分类是检索标签，不代表器件适用性判断或硬件选型结论。

## 更新索引

在仓库根目录运行：

```powershell
python 知识库/tools/build_pdf_index.py
```

脚本会重新扫描全部 PDF，提取页数、元数据和前 3 页文本，计算 SHA-256，并重建 Markdown、CSV 和 JSON 索引。脚本不会写入任何原始 PDF。

依赖：Python 3 和 `pypdf`。若默认 Python 未安装依赖，可使用 Codex 工作区自带的 Python 运行。

## 边界

- 原始目录、压缩包、源码、图片、视频和 Office 文件均保留原样。
- 本轮只为 PDF 建立逐文件索引；非 PDF 资料仅在审计中记录其存在与重复概况。
- 未联网核对厂商官网或竞赛官网，因此“最新版本”和“缺失资料”均以本地目录为证据，需要后续用官方来源复核。
- 本知识库不做硬件选型。只有在收到明确选型需求、约束和评价标准后，才可以另行建立对比与决策记录。
